# Sequence accession IDs used in differenet analyses

## Note
- Accession_ID_denvX_global_even: for global phylogenetic and phylogeographic analyses under spatially even sub-sampling scheme.
- Accession_ID_denvX_global_infection: for global phylogenetic and phylogeographic analyses under infection-informed sub-sampling scheme.
- Accession_ID_denvX_temspa: for the estimates of viral migration among spatiotemporal demes.
- Accession_ID_denvX_China: for the analyses in China.
- Accession_ID_denvX_GLM: for GLM phylogeographic inference at the country level.
