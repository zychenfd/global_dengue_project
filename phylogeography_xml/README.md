# phylogeography_xml_template

## Note
- BEAST1_global_template: for global phylogenetic and phylogeographic analyses.
- BEAST1_temspa_template: for the estimates of viral migration among spatiotemporal demes.
- BEAST1_china_template: for the analyses in China.
- BEAST1_GLM_template: for GLM phylogeographic inference at the country level.
